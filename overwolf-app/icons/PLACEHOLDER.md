# Icons

Overwolf expects three icons in this folder:

- `icon-256.png` — 256x256 PNG, used in the Overwolf dock.
- `icon-256-gray.png` — same icon, grayscale (shown when the app is paused).
- `icon-256.ico` — 256x256 Windows .ico, used in the system tray.

Drop in any three matching files before you pack the `.opk`. They are not
committed because they're branding-specific.
